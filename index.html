Amazon WebScrapping

This Python project implements a web scraper to track Amazon product prices. It utilizes requests to fetch product pages and BeautifulSoup to parse the HTML. Extracted title, price, and date are stored in a CSV file using csv and analyzed with Pandas. The script, check_price, periodically checks for price changes and can trigger email alerts (using smtplib) when a desired price is reached (function not implemented for security reasons). This project demonstrates proficiency in web scraping with libraries like requests, BeautifulSoup, csv, and pandas.
